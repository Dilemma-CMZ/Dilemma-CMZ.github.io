2019 Multi-University Training Contest 1 Operation

发表于 2019-07-23 | 更新于 2019-07-25
本文字数： 9.4k | 阅读时长 ≈ 9 分钟

题目链接 Operation

Description

有一个序列，2种操作

  1. 查询$l,r$中，任取几个数使得异或和最大.输出这个max
  2. 在序列末尾插入一个新的数字，并使$n=n+1.$

要求强制在线

Solution

首先这道题目显然会想到线性基。
暴力的做法可以用数据结构维护区间线性基，比如分块$\mathcal O(n\log^2n)$，线段树$\mathcal O(n\log^3n)$或者$ST$表$\mathcal O(n\log^3n)$，考场上我们全部都用过一遍，但肯定过不了.

然后我们就可以思考一下，是否有什么可以优化的办法
我们可以考虑贪心，对于每一个位置的前缀线性基，我们在$add$的时候，让最高位的数字的位置尽量偏右，这样就不需要维护区间的线性基了，毕竟线性基合并就已经两个$log$了.

至于正确性应该显然吧，因为我们优先考虑高位，其次再考虑位置，这样肯定保证了最高位的最优决策性。
